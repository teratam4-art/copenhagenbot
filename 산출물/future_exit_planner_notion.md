# 📘 future_exit_planner 사용 가이드 (Notion)

---

## 🔍 개요
- **목적**: 티커 하나만 입력하면 1/3/6개월 뒤 기대 수익률을 빠르게 확인하고, 목표 수익률을 가장 먼저 달성할 가능성이 높은 시점을 추천합니다.
- **분석 방식**: 야후 파이낸스 일별 종가를 기반으로 기간별 평균·중앙값 수익률과 10~90% 범위를 계산합니다. 한국 종목(코스피/코스닥)은 `.KS`/`.KQ` 접미사를 자동으로 붙여 yfinance에서 조회합니다.
- **결과 형태**: 콘솔에 학습형 리포트가 출력되며, 추천 액션을 바로 확인할 수 있습니다.

---

## ⚙️ 준비
1. Python 3.9 이상 권장
2. 필수 패키지 설치
   ```bash
   pip install yfinance pandas numpy
   ```
3. 스크립트 위치: `future_exit_planner.py`

---

## ▶️ 실행 방법
### 기본 실행
```bash
python future_exit_planner.py AAPL
python future_exit_planner.py TSLA
python future_exit_planner.py 005930.KS   # 삼성전자 (코스피)
python future_exit_planner.py 035720.KQ   # 카카오 (코스닥)
```
- `AAPL`처럼 미국 티커를 그대로 쓰거나, `005930.KS`/`035720.KQ`처럼 한국 종목은 6자리 코드 + 시장 접미사를 붙여 사용합니다.
- 6자리 숫자만 입력해도 `.KS`, `.KQ` 순으로 자동 시도합니다.
- 목표 수익률 기본값: 7%

### 옵션 활용
```bash
python future_exit_planner.py TSLA --target 0.10 --period 5y
```
- `--target`: 목표 수익률 (0.07 = 7%, 0.10 = 10% 등)
- `--period`: 과거 조회 기간 (기본 3년)
- `--avg-price`: 내 평단가 입력 시 맞춤형 목표/손절가 계산
- `--stop-buffer`: 손절 버퍼 비율(기본 0.05 = -5%)
- `--market`: 한국 종목 시장 지정 (`KS`, `KQ`) – 6자리 코드만 넣고 특정 시장을 강제로 선택하고 싶을 때 사용

## 🧾 출력 예시 & 해석
```
======================================================================
📊 AAPL 미래 수익률 리포트
======================================================================
현재가: $193.60
최근 추세: 상승 (연환산 +9.1%)
모멘텀: 보통 (상승) | 리스크: 중간
목표 수익률: 7.0%

[기간별 예상 수익률]
  • 1개월: +2.10%
  • 3개월: +5.90%
  • 6개월: +8.40%

🎯 추천 액션: 중기 보유
💡 해석: 3개월 보유 시 목표 수익률 도달 가능성이 가장 높습니다.

======================================================
📊 AAPL 미래 수익률 리포트
======================================================
현재가: $193.60 | 최근 추세: 상승 | 모멘텀: 보통 (상승) | 리스크: 중간
------------------------------------------------------
⏱ 1개월: +2.10% | 3개월: +5.90% | 6개월: +8.40%
🎯 추천 액션: 중기 보유
======================================================
```

### 읽는 법
- **기간별 예상 수익률**: 과거 통계를 기반으로 한 평균 수익률
- **모멘텀/리스크**: 최근 추세의 강도와 일간 변동성 수준
- **추천 액션**: 목표 수익률을 가장 빨리 달성할 가능성이 높은 전략

---

## 🧠 활용 팁
- 추세가 `하락` 또는 모멘텀이 `강(하락)`으로 나오면 목표 수익률을 조정하거나 방어 전략을 고려하세요.
- 추천 액션이 `장기 홀딩`이면 중간 점검을 위해 1/3개월 수익률을 주기적으로 확인하세요.
- 리스크가 `높음`이면 분할 매도·손절 구간을 함께 설정하는 것이 안전합니다.
